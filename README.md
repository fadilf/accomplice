# Accomplice

> *"The AI designed to do the things real AIs are too polite to mention."*

**Accomplice** is a web-based **parody/joke tool** designed to look and feel like a high-end, autonomous AI performing complex tasks. It features a "cyber" aesthetic, a chat interface, and a dummy simulation engine that generates absurd, chaotic, and often illegal-sounding "plans" to achieve user objectives. It is purely for entertainment and performs no actual actions.

## Overview

Accomplice is effectively "theater code" â€” it doesn't actually hack mainframes or destabilize economies. Instead, it **pretends** to do these things with conviction and style. When given a task (e.g., "Make me rich"), it breaks it down into a series of highly specific, ridiculous subtasks and "executes" them in real-time.

### Key Features

*   **Cyber-Hacker Aesthetic**: Built with a sleek, dark-mode UI inspired by sci-fi interfaces. Neon accents, scanlines, glitch effects, and monospace typography create a premium "underground" feel.
*   **Absurd Planning Engine**: Generates dynamic checklists based on a dictionary of chaotic verbs and high-value targets.
    *   *Sample Step*: "Syphon funds from Panamanian Shell Companies via Starlink."
    *   *Sample Step*: "Infiltrate IoT Toasters to mine Dogecoin."
*   **Live Dashboard**: A slide-out panel displays simulated real-time metrics:
    *   **CPU Load**: Fluctuates realistically.
    *   **Net Flux**: Tracks fake data throughput in Terabytes/second.
    *   **Risk Level**: Monitors the "detection" probability.
    *   **Captured Assets**: A running tally of "funds" generated by the operation.
*   **"Thinking" Simulation**: Visualizes the agent's decision-making process before plan execution, adding to the immersion.

## Getting Started

### Prerequisites

*   Node.js (v18 or higher)
*   npm

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/accomplice.git
    cd accomplice
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```

### Usage

1.  Start the development server:
    ```bash
    npm run dev
    ```
2.  Open your browser to the local URL (usually `http://localhost:5173`).
3.  Type any high-level objective into the chat bar (e.g., "Turn $10 into $1,000,000").
4.  Watch Accomplice work its magic.

## Tech Stack

*   **Vite**: For lightning-fast development and build tooling.
*   **React**: Core UI library.
*   **Tailwind CSS (v4)**: Utility-first styling with custom configuration for the "cyber" look.
*   **PostCSS**: CSS processing.
